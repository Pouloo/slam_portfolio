E6

Chers membres du jury, vous trouverez sur cette page les documents requis pour le déroulement de l’épreuve E6 de l’examen du BTS SIO SLAM : les fiches descriptives de deux réalisations professionnelles ainsi que les éléments constitutifs de ces réalisations. Des liens vers les répertoires Git correspondants sont également mis à votre disposition pour l'examen du code source.

Réalisation 1

Le projet THON MAYO est un site d’e-commerce, développé pour une entreprise fictive agissant dans l’industrie agroalimentaire. Cette activité à été réalisée en groupe, au cours de la formation scolaire. L’accès à la fiche descriptive de la réalisation professionnelle est protégé par un mot de passe (communiqué en amont aux membres du jury) en raison des informations sensibles qui peuvent figurer sur le document.

Projet : THON MAYO
Lieu de réalisation : En classe
Type de projet : Site d’e-commerce

Le code source de ce projet est disponible en intégralité sur GitHub.com

Réalisation 2

Le projet FULLFORT CROUSTY est un site de commandes en ligne pour des produits fast-food. Il a été développé sous forme de prototype dans le cadre d’un stage au sein de l’entreprise FULLFORT PRODUCTION. Le rapport de stage en lien avec cette expérience peut être consulté dans la partie E5 de ce portfolio. L’accès à la fiche descriptive de cette réalisation professionnelle est également protégé par un mot de passe (communiqué en amont aux membres du jury) en raison des informations sensibles pouvant s’y trouver.
